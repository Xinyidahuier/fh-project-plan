FlashHome 核心业务流程图

主线流程 + 商家入驻 / 网点注册 / 商品审核 / 售后退款 共9条业务流

1 主线交易流程
买家点击推广链接 → 商品详情页 → 下单（COD / PromptPay） → 卖家发货 → Flash物流配送 → 买家确认收货
2 网点推广流程
网点登录H5 → 查看今日推荐 → 一键分享（LINE / FB / 海报） → 买家通过链接下单 → 网点获得佣金
3 推广日历流程
FH后台配置日历 → 定时推送（10 / 13 / 18点） → 网点收到提醒 → 一键转发至社群 → 触达买家
4 商家入驻流程
商家提交入驻申请 → FH后台审核资质 → 通过 → 开通卖家后台 → 上架商品 → 商品进入推荐池
5 网点注册流程
FH后台生成邀请链接 → 定向发给目标网点 → 网点填写信息 → 账号激活
6 商品审核流程
卖家上传商品 → FH后台审核 → 通过 → 上架 / 驳回 → 卖家修改重新提交
7 结算流程
订单确认收货 → T+7 生成结算单 → 人工审批 → 打款至卖家 / 网点 → 完成结算
8 售后退款流程
买家发起售后 → LINE群通知客服 → 审核（仅退款 / 退货退款） → 退货走Flash物流 → 退款到买家
9 支付流程
买家提交订单 → 选择支付方式 → PromptPay QR扫码 / COD 货到付款 → 支付完成
